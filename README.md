# Arduino-Based-Distance-Warning-Prototype
Uses an HC-SR04 ultrasonic sensor along side an imported SR04 library to estimate the distance between the sensor and an obstacle. The program has 7 different warning states which depend on the distance the ultrasonic sensor calculates. LEDs ranging from green to yellow and red along side an active buzzer act as a warning system to inform the user of their proximity to an object.

The distance values have been set to mimic a car's parking sensor.

Required Materials:
    -Arduino Uno
  -Breadboard
  -6 resitors 330ohm recommended
  -6 LEDS (3 different colors idealy)
  -Active buzzer
  -Jumper wires
